# AozoraChecker

AozoraBunko character checker, based on checker.cgi.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'aozora_checker'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install aozora_checker

## Usage

### CLI (command line interface)

```
$ aozora-checker foo.txt
```

### Apps

```
$ foreman start
```
